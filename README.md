## Descrição

Repositório referente aos meus estudos do Framework NestJs

## Estudos:

- Trabalhando com controllers e services
- Parâmetros de requisição
- Customizando status code
- Manipulando requisições, Get, Post, Update, Delete
- Módulos do NestJs
- Tratamento de erros com Nest

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
